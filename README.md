# FileMate

A user-friendly Python tool to automatically sort files based on type and content.

## Features
- Automatically sort files by extension
- Optional content-aware sorting for text, images, and more
- Real-time folder watching
- Undo last move and logging

## Installation
```bash
git clone https://github.com/dhanushka-sys/FileMate.git
cd FileMate
pip install -r requirements.txt
